# Role to Skill Alignment

This reference maps the resume registry roles to the strongest supporting capability areas from the skills and experience inventory.

## Forward Deployed Engineer

Primary themes:
- Customer-embedded technical delivery
- Ambiguous problem discovery and rapid scoping
- AI workflow customization and operationalization
- Integrations, telemetry, and production rollout
- Product feedback loops and adoption acceleration

Best supporting capability areas:
- Azure AI and agentic workflows
- Observability and diagnostics
- Escalation leadership
- Operational playbooks and automation
- Executive and stakeholder communication

## Platform Engineer

Primary themes:
- Internal platform enablement
- Self-service automation
- Observability and reliability foundations
- CI/CD and infrastructure as code

Best supporting capability areas:
- Bicep and infrastructure automation
- Azure DevOps pipelines
- Operational metrics and KQL dashboards
- Service reliability and alert rationalization

## Build Engineer

Primary themes:
- Build and release automation
- Deployment safety
- Pipeline governance
- Change quality and release repeatability

Best supporting capability areas:
- Azure DevOps
- Git-based release flows
- PowerShell automation
- Production readiness and runbooks

## Director Incident Manager

Primary themes:
- High-severity incident command
- Executive advocacy
- Crisis communication
- Global operations leadership
- Service recovery and post-incident improvement

Best supporting capability areas:
- Incident management
- Root cause analysis
- Signal reduction and alert tuning
- Global team leadership

## Technical Program Manager

Primary themes:
- Cross-functional strategy and execution

Best supporting capability areas:
- Program planning
- Telemetry and reporting

## Senior Site Reliability Engineer

Primary themes:

Best supporting capability areas:

## Staff Escalation Manager

Primary themes:
