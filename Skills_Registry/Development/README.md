# Development Skills Registry

This directory contains skills for general software development, project setup, code quality, and development tools.

## Available Skills

### 1. **agent-customization** ✅
**Purpose**: Create, update, review, fix, or debug VS Code agent customization files  
**When to Use**: Saving coding preferences; troubleshooting instructions; configuring agent modes  
**Key Features**:
- Customization file management (.instructions.md, .prompt.md, .agent.md)
- YAML frontmatter handling
- Tool restrictions
- Custom workflows

### 2. **cosmosdb-best-practices** ✅
**Purpose**: Azure Cosmos DB performance optimization and best practices  
**When to Use**: Writing Cosmos DB code; designing data models; optimizing queries  
**Key Features**:
- Performance guidelines
- Partitioning strategies
- Query optimization
- SDK best practices

### 3. **get-search-view-results** ✅
**Purpose**: Get the current search results from the Search view in VS Code  
**When to Use**: Accessing search results; integrating with search workflow  
**Key Features**:
- Search result retrieval
- VS Code integration

### 4. **project-setup-info-local** ✅
**Purpose**: Comprehensive setup steps to create complete project structures  
**When to Use**: Creating new projects; setting up frameworks; initializing scaffolding  
**Key Features**:
- Project initialization
- Framework setup
- Dependency configuration
- Development environment setup
