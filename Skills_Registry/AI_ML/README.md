# AI & ML Skills Registry

This directory contains skills for AI/ML development, model optimization, agent building, and AI-powered workflows.

## Available Skills

### 1. **foundrytk-quick-start** ✅
**Purpose**: Quick-start guide for Foundry Toolkit (formerly AI Toolkit) in VS Code  
**When to Use**: Get started with Foundry Toolkit; explore features; build agents; manage toolboxes  
**Key Features**:
- Setup guidance
- Model exploration
- Agent building
- Toolbox management
- Interactive tutorials

### 2. **microsoft-foundry** ✅
**Purpose**: Deploy, evaluate, fine-tune, and manage Foundry agents end-to-end with azd  
**When to Use**: Create agent; hosted agent scaffold/run/deploy; evaluate agent; optimize prompt  
**Key Features**:
- Agent deployment
- Hosted agent management
- Performance evaluation
- Continuous evaluation
- Prompt optimization
- Model fine-tuning (SFT/DPO/RFT)
- Agent Optimizer
- Dataset curation from traces

### 3. **use-winml-cli** ✅
**Purpose**: Build, optimize, quantize, compile, and benchmark ONNX models for Windows ML  
**When to Use**: Run models on Windows AI PC; target NPU acceleration; prepare for on-device inference  
**Key Features**:
- Model optimization
- Quantization
- Compilation
- Benchmarking
- NPU targeting (Qualcomm QNN, Intel OpenVINO, AMD VitisAI)
- GPU & CPU execution
- Hugging Face model support

### 4. **vscode-microsoft-foundry** ✅
**Purpose**: End-to-end workflow for developing AI agent apps on Microsoft Foundry  
**When to Use**: Build agent; scaffold agent; create hosted agent; deploy agent; evaluate agent  
**Key Features**:
- Agent development lifecycle
- Application scaffolding
- Deployment workflows
- Performance measurement
- Production deployment

## Learning Path

### Beginner
- Start: `foundrytk-quick-start`
- Next: `vscode-microsoft-foundry`
- Then: Build simple agent

### Intermediate
- Study: `microsoft-foundry` best practices
- Focus: Agent optimization and evaluation
- Practice: Multi-agent workflows

### Advanced
- Master: Fine-tuning (SFT/DPO/RFT)
- Explore: `use-winml-cli` for optimization
- Build: Production-grade agents with custom models
